Здесь разместил некоторые мои наработки на JAVA тестирования

Везде используется Maven и TestNG

api - использую rest-assured

cucumber - подход тестиование через BDD

dbapi - совместно rest-assured и JDBC

mobiluialfa - тестирование мобильного приложения на устройстве Андройд, используя Appium + Android Studio

robotframework - подход тестиование через BDD, тоже самое что и cucumber, только всё на английском

springdbapi - rest-assured и JDBC используя фрейворк spring

ui - используется selenium и cromedriver

Все тесты рабочие. Могут не работать из-за зависемостей, которые пересекают подходы. Нужно некоторые комментировать, чтоб некоторые тесты сработали.
